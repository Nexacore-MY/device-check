# Nexacore Brand Guidelines

## Colours
| Name | Hex | Usage |
|------|-----|-------|
| White | #FFFFFF | Backgrounds, text on dark |
| Ice | #CEF1F2 | Light accent, highlights, card backgrounds |
| Teal | #44C0C5 | Primary accent, interactive elements |
| Deep Teal | #117F84 | Secondary accent, hover states |
| Dark | #06444A | Primary dark, headers, text |

## Typography
- **Montserrat** — Headings
- **Quicksand** — Body text
- **Poppins** — Buttons

## Components
- Buttons: pill-shaped (full border-radius)
